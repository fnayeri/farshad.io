Platform Architect
Interactive Circuit Diagnostic System
Verizon Labs

Led team to design & develop a unified platform for testing all Verizon Circuits.

telecomm full stack api platform native apps

Problem

Verizon field technicians used to depend on help from central NOC operators for their diagnostic and maintenance activities. These requests would queue, keeping costly field technicians to idle.

Stakes

In order to address the broad set of uses at Verizon, the diagnosis system had to accommodate many circuit types (T1, FFTP, DDS, DSL, POTS, …). All this had to be done within Verizon’s carrier-grade standards of reliability. The native apps had to run in network closets and basements where there would be no wi-fi or cellular access.

Discovery

An instant replay mechanism was devised to accommodate at the same time: case review, as well as dealing with network disruptions due to poor or non-existent connection.

Outcome

The system was deployed internally to all Verizon field technicians, in some cases reducing idle time by hours. Subsequent versions of the system was shipped to support a wide range of circuit types.
