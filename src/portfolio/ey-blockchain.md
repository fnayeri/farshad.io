Led international agile team to plan, define & develop EY Global Innovations enterprise blockchain platform based on a federated microservice architecture on top of Azure cloud.

## Problem
Augment EY's consulting business with a family of subscription services from EY Global Innovation.

## Stakes
As a global services company, EY has many subsidiary businesses with conflicting interests. A new platform had to be available to both existing and new applications, based on Azure Cloud.

## Discovery
The legacy system tightly integrated front-end applications with the API platform. The new platform needed to decouple this into a set of microservices around core platform services such as authentication and metering.

## Outcome
Successfully launched a robust blockchain API platform, an accompanying website, and a full launch and migration plan.
